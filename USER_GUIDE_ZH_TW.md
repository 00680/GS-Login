# GSLogin — 使用指南（繁體中文）

本指南說明如何在 GSLogin GUI 中設定帳號、遊戲資料夾，並啟動自動登入流程。

1. 帳號（帳號頁面）
- 開啟應用程式的帳號頁面。
- 點選「新增」或「Add / New」以建立新帳號項目。
- 填寫以下欄位：
  - 使用者名稱：帳號名稱或電子郵件。
  - 密碼：帳號密碼。
  - OTP / 兩步驗證：選填 — 僅在 TW 伺服器需要，請輸入一次性驗證碼（若適用）。
  - 伺服器：從清單選擇目標遊戲伺服器。
- 儲存帳號。可以新增多個帳號，並稍後編輯或刪除。

2. CapSolver API Key（設定頁面）
- 開啟設定（Settings）頁面。
- 找到 CapSolver 或驗證碼解決服務的 API Key 欄位。
- 貼上你的 CapSolver API Key 並儲存設定。此項設定可在遇到驗證碼時自動使用服務解決。

3. 遊戲資料夾（Process 頁面）
- 開啟 Process 頁面（或管理遊戲資料夾/程序的頁面）。
- 使用「新增資料夾」或瀏覽按鈕，開啟檔案對話方塊並選擇本機遊戲安裝資料夾。
- 每個選擇的資料夾會列在清單中；可新增多個資料夾。

4. 啟動登入自動化
- 在 Process 頁面，從資料夾清單選擇目標遊戲資料夾。
- 從帳號選擇器選取已設定的帳號。
- 按下 Run / Start 按鈕以啟動自動登入流程。
- 程式會啟動遊戲或瀏覽器並使用所選的帳號與資料夾嘗試登入。請注意日誌或畫面狀態以查看進度與錯誤。

注意事項
- 僅對 TW 伺服器需要 OTP 時，請在建立或編輯帳號時填入當前的一次性驗證碼。
- 若預期會遇到驗證碼，請確保 CapSolver API Key 有效；否則可能需要手動處理驗證碼。
- 若需要 Playwright 的瀏覽器執行檔，請依照建置說明或腳本準備 `PLAYWRIGHT_BROWSERS_PATH`。

故障排除
- 若登入失敗，檢查應用程式日誌並確認帳號、伺服器選擇與資料夾路徑正確。
- 檢查網路連線與任何可能阻擋自動瀏覽器控制的防毒或系統限制。
